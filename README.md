# Student Performance Analysis

## Project Overview
This project explores the factors influencing student academic performance using Exploratory Data Analysis (EDA).

## Objectives
- Understand the dataset structure.
- Identify factors associated with final grades.
- Discover actionable insights for educational institutions.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings
- Previous grades (G1 and G2) were the strongest predictors of final grades.
- Previous failures showed a moderate negative relationship with academic performance.
- Higher study time categories generally achieved higher grades.
- Mother's education showed a stronger relationship with grades than father's education.
- Health and alcohol consumption showed little relationship with final grades.

## Dataset
UCI Student Performance Dataset
